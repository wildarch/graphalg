# Playground Stories
1. Write GraphAlg programs with syntax highlighting.
2. Show the output matrix of a custom GraphAlg program, with predefined input matrices.
3. Show error diagnostics on compile errors inside the editor view.
4. Show GraphAlg core IR generated from a GraphAlg program

## Hooks on the Editor
- On edit: remove error diagnostics.
- On save: automatically run the compiler after some delay?

## Main Interface Buttons
- Compile (foldable output)
- Run GraphAlg program
- Turn VIM mode on/off
